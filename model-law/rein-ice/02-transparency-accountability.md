## Why Transparency Matters

**For Communities:**
- Know who is conducting enforcement in your area
- Document civil rights violations
- Enable legal accountability

**For Local Government:**
- Track resource use on federal immigration enforcement
- Ensure compliance with local policies
- Maintain community trust

**Advocacy Opportunities:**
- FOIA/public records requests for ICE activity data
- Ordinances requiring notification of federal enforcement
- Officer identification requirements for all law enforcement
- Public reporting on detention and deportation statistics

## California SB 627 - Agent Identification Requirements

**Jurisdiction:** California (statewide)
**Legal Citation:** SB 627
**Type:** Enacted law
**Requires:** Identification mandate, Notification/Reporting

**Key Issue:** FBI has documented crimes by impersonators posing as ICE agents. Identification requirements protect communities and support legitimate law enforcement.

## Using Transparency as an Organizing Tool

**For Individuals:**
1. Know what identification to request (badge number, warrant, agency)
2. Document all encounters (video if safe, written notes)
3. Report incidents to local immigrant rights organizations
4. File complaints for violations

**For Employers/Businesses:**
1. Post know-your-rights information
2. Train staff on protocols
3. Require warrants for access
4. Document and report enforcement visits

**For Advocates:**
1. Track local enforcement patterns
2. File FOIA requests for data
3. Push for local notification ordinances
4. Support federal transparency legislation
